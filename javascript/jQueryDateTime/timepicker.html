Notes

  • Cannot use the time slider with finger. You can select a point on the slider with finger, but there is a minor bug meaning you need to select twice to get the time to register. Same issue if you try with the mouse
  • You can edit the bound field while the widget is active
  • Generally a very neat extension of the standard datepicker. Doesn't support all features however e.g. alternate fields (note above - the alternate field only gets the date, not the time)
